# 🌐 Multi-Router Network with Static Routing

## 📌 Overview

This project demonstrates communication between two different office networks using static routing and a central server.

## ⚙️ Configuration

* Configured two routers with LAN and WAN interfaces
* Implemented static routing between networks
* Connected multiple PCs across two offices
* Configured a server with HTTP service

## 🧪 Testing

* Verified connectivity using ping
* Accessed server using web browser
